# Java命名规范

**只是建议**

1. 类和接口名称：每一个单词首字母大写，例如：TestDemo;

2. 变量/方法名称：第一个单词首字母小写，之后每个单词的首字母大写，例如：studentName;printInfo();【驼峰原则】

3. 常量名称：每个字母大写,若有多个单词，以_分割，例如：FLAG，MAX_VALUE;

4. 包名称：所有字母小写

   ​       多级包： 域名反转，例如：www.zju.edu.cn   -->  cn.edu.zju
